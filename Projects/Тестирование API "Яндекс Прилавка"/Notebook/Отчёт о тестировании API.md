## Отчет о тестировании Яндекс Метро
Тестирование проводилось в следующем тестовом окружении:HOST https://e61a145b-9e44-405b-b7f0-1b40ae171445.serverhub.praktikum-services.ru в приложении Яндекс Прилавок. Все требования, затронутые изменениями, были покрыты чек-листом: (https://docs.google.com/spreadsheets/d/1ZkoMjjRMpvXI6dhqRB7OWDAQ2vGVgmidvjM3a_iCfog/edit?usp=sharing ).
Результаты выполнения тестов можно посмотреть здесь: (https://docs.google.com/spreadsheets/d/1G3b1iAKThHgM4Sfo8YtbOk-nVOj2hkqJchKalylnAkw/edit?usp=sharing ).
Из (243 проверок) успешно прошло (201 успешных тестов), не прошло — (42 не пройденных тестов).

Затрачено времени было на написание тестовой документации 24 часа, и тестирование заняло 10 часа. При проверке приложения удалось применить функциональное тестирование. После исправления багов стоит перепроверить тесты.
С учётом того, что найдены критические дефекты располагаются в пользовательском сценарии команда тестирования против публикации новой функциональности приложения Яндекс Прилавка.
