## Краткое описание проекта

Яндекс Метро — нативное мобильное приложение для навигации в метрополитене. Позволяет строить маршруты между станциями, получать информацию о времени в пути, пересадках, событиях на станциях (закрытия, ремонты), а также использовать карту метро с возможностью масштабирования и перемещения.

## Исследуемый набор данных

* Схемы метро 36 городов (включая Москву, Санкт-Петербург, зарубежные города).

* Данные о станциях: название, линия, координаты, события (закрыта, ремонт и т.д.).

* История маршрутов и выбранных станций (хранится локально).

* Настройки пользователя: город, язык, тема, избранное.

## Основные пункты и цели исследования

* Проверка корректности построения маршрутов при различных комбинациях станций.

* Тестирование работы геолокации: определение ближайшей станции при запуске, поведение в городах с метро и без.

* Проверка интерфейса при разных ориентациях устройства (портретная/альбомная) и разрешениях экрана.

* Тестирование жестов: тап, лонг-тап, свайп, пинч, спред.

* Проверка работы при отсутствии интернета (авиарежим).

* Тестирование настроек: смена города, языка, темы (светлая/тёмная/авто), очистка истории.

* Проверка сохранения состояния интерфейса при повороте экрана.

* Тестирование обновления приложения и обратной совместимости.

## Используемые библиотеки и инструменты

* Pairwise Testing 

* Postman — тестирование бэкенда
